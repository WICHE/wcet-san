    dependent-selector

    @mixin dependent-selector($selector, $boolean) { ... }
    View source

    Description

    Wrap @content in $selector if true, else just print @content. This is useful for situations like when you need to be able to use a button mixin on both button/a elements and input elements, but the mixin requires pseudo elements which means the input will need to be wrapped in a span.

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $selector

    A valid CSS selector

    		String— none
    $boolean

    A boolean representing whether or not the selector should be used

    		Boolean— none

    Content

    This mixin allows extra content to be passed (through the @content directive).

    Example

    @include dependent-selector('input[type="submit"]', true) {
  appearance: none;
  line-height: 1;
}
